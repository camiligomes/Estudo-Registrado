Unidades

A Fetch API e o XMLHttpRequest (XHR) são duas abordagens para realizar solicitações de rede em JavaScript, principalmente para recuperar dados de recursos externos, como arquivos, APIs ou servidores web. Ambos são usados para comunicação assíncrona, o que significa que você pode fazer solicitações e receber respostas sem bloquear a execução do restante do código JavaScript. Em resumo, a Fetch API é a escolha preferida em desenvolvimento moderno devido à sua interface mais limpa, uso de Promessas e melhor suporte a formatos de dados comuns. O XMLHttpRequest ainda é usado em cenários legados e quando a compatibilidade com navegadores mais antigos é importante.
